# Learning-algorithm
### 前提
> 算法虐我千百遍,我待算法如初恋.

算法一直是本人心中的一个梗。   

作为一个科班出生的码农，在最应该好好学习的大学时代并没能很好地掌握这些算法中的精髓。   

### 背景
> 程序 = 数据结构 + 算法   

工作到现在，在实际工作中并不会怎么用到算法，更多的是处理业务逻辑和堆砌API。但是我一直认为算法和数据结构是一个合格码农的基本功。算法、数据结构是内功，技术是招式。   

### 目的
重新捡起基础的算法和数据结构，夯实基础，平复日渐浮躁的内心。随便在缅怀下眨眼即逝的大学时光。
